Внесенные изменения (на 04.06.2003):
Изменения:
        -Команда special для скаута включает/выключает автоскан
        автоскан производится раз в секунду
        радиус 50 или последний использовавшийся
	- Spys can now silently feign without a death scream with the command "sfeign".
		The feign command in the spy's "special" menu will only do the default
		non-silent feign.(tf 2.9)
	- When a spy feigns he will now drop all goal items.(tf 2.9)

	- Добавлена новая возможность настройки через битовую маску
	 setinfo s <число>
	 1 - exec map on
	 2 - exec class on
	 4 - class help on
	 8 - autoid on
	- Made abbreviations for setinfo commands:
	  1 = disc_s 
	  2 = disc_n
	  3 = disc_r
	  4 = disc_c
	  пустая строка означает стандартный дискард для данных боеприпасов
Bugs fixed:
	  -server crashed.. при ID на сентри

Внесенные изменения (на 27.05.2003):

Cкаут.
[+] Мертвый скаут не может снимать детпаки (TF 2.9).

Демоман.
[+] Добавленна отдельная модель для пайп-лаунчера (TF 3.0).
[+] В 1,5 раза увеличено минимальное допустимое расстояние между двумя детпаками.
[+] Время до взрыва детпака выводится в статус бар.

Медик.
[+] Убран баг с лечением флэша (TF 2.8.1 b1).
[+] Получает желтый армор сразу при рождении.

Пиро.
[+] Убран баг с падением болванки напалма (TF 2.8.1 b1).

Спай.
[+] С фейнутого спая не берется "id" (TF 2.9).
[+] По фейнутому спаю не стреляет сентри (TF 2.9).
[+] Спай может фейнится в воздухе (OmiTF).

Инженер.
[+] Исправлен алгоритм наведение сентри (tf 2.8.1 b1).
[+] Возможнось выбора между несколькими алгоритмами наведения: default (TF
2.8.1 b1), localinfo sg old (TF 2.8.1), localinfo sg fix (OmiTF).
[+] Получает желтый армор сразу при рождении.
[+] Статус бар демонстрирует здоровье сооружений (Dragon TF).
[+] Команда "id" показывает здоровье, патроны и ракеты у любой из сентри
своей команды.
[+] Добавленно сообщение о взрыве диспенсера.
[+] EMP не взрывает постройки своей команды, если на сервере включен френдли
файр (tf 2.9).

Общее.
[+] Встроенный autoid, включается "setinfo autoid on" (делает id раз в секунду, вкл/выкл срабатывает при респавне).
[+] Настраиваемый дискард 
    "setinfo disc_s <сколько оставить>"
    "setinfo disc_n <сколько оставить>"
    "setinfo disc_r <сколько оставить>"
    "setinfo disc_c <сколько оставить>"
[+] Убран баг пропадания последней гранаты, при взятии goal_item.
[+] Аптечки лечат повреждения ног, даже если здоровье у игрока больше или равно максимальному (TF 2.9).
[+] Убрана проблема возникающая при взятии армора из goal_item (TF 2.9).

Данная версия не является окончательной. Планируется внесения еще ряда дополнений.
